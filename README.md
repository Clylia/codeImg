# 在线文本转图片工具

## 工具简介
### 技术栈
基于 `react` + `react hook` + `codemirror` + `html2canvas` + `flygrace UI` 等开发一款PC端文本转图片的工具

## 使用地址
- 😋 工具使用地址：http://halodong.gitee.io/codeimg/

## 功能
### 核心功能
- 截图功能：
  - 支持超长截图
- 高定制化：
  - 可自定义文本大小
  - 可自定义相框大小
  - 可自定义相框颜色
- 文本复制：
  - 可复制输入框文本

### 规划中
- 拖拽改变输入框大小
- 更高清晰度
- 以及更多自定义功能，可以在[github](https://github.com/halodong/codeimg)留言
